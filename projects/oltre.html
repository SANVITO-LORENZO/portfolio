OLTRE

Superando le barriere digitali

Il Progetto

Oltre è un'estensione browser progettata per rendere il web accessibile a tutti. Analizza la struttura delle pagine in tempo reale per ottimizzare contrasti, dimensioni dei caratteri e leggibilità per utenti con disabilità visive o cognitive. Oltre is a browser extension designed to make the web accessible to everyone. It analyzes page structures in real-time to optimize contrasts, font sizes, and readability for users with visual or cognitive disabilities.

Tech Stack

  • JavaScript (ES6+)
  • Chrome API
  • CSS Variable Injector

Core Features

Smart Contrast: Adattamento dinamico dei colori basato sulle linee guida WCAG.

Dyslexia Friendly: Integrazione di font specifici e spaziatura ottimizzata.
